Java Script

¿Qué es JS?

Lenguaje de programación ligero, interpretado, o compilado justo-a-tiempo (just-in-time) con funciones de primera clase. Si bien es más conocido como un lenguaje de scripting (secuencias de comandos) para páginas web, y es usado en muchos entornos fuera del navegador, tal como Node.js, Apache CouchDB y Adobe Acrobat JavaScript es un lenguaje de programación basada en prototipos, multiparadigma, de un solo hilo, dinámico, con soporte para programación orientada a objetos, imperativa y declarativa (por ejemplo programación funcional).
Es un lenguaje fácil de aprender pero difícil de dominar y se utiliza para una amplia variedad de propósitos, desde simplemente mejorar la funcionalidad de un sitio web hasta ejecutar juegos geniales y software basado en web.
La sintaxis básica es similar a Java y C++ con la intención de reducir el número de nuevos conceptos necesarios para aprender el lenguaje. Las construcciones del lenguaje, tales como sentencias if, y bucles for y while, y bloques switch y try ... catch funcionan de la misma manera que en estos lenguajes (o casi).

¿Cómo funciona?

Dentro de una página web normal, que contiene HTML y CSS, se coloca código JavaScript. Cuando el navegador carga la página, tiene un intérprete incorporado que lee el código javaScript que encuentra en la página y lo ejecuta. El intérprete del navegador integrado es básicamente la "máquina virtual JavaScript". Diferentes motores tienen diferentes nombres en clave..
El JavaScript moderno es muy potente en comparación con las versiones lanzadas hace 10 años.En el navegador, javascript puede hacer muchas cosas, como interactuar con el usuario, manipular la página web y el servidor web. Algunos de estos son:

  • Agregar nuevo contenido HTML a la página
  • Reaccionar a acciones del usuario como clics del mouse, movimientos del puntero, etc.
  • Puede obtener y configurar cookies
  • Remember the data on the client side.

Hay varias cosas que JavaScript NO puede hacer y algunas de ellas pueden estar relacionadas principalmente con el principio de seguridad del "usuario".

  • JavaScript en el navegador no puede leer/escribir archivos arbitrarios en el disco duro, copiarlos ni ejecutar programas. Básicamente, no tiene acceso directo a las funciones del sistema operativo.
  • No puede proteger los recursos o las imágenes de su página.
  • No puede acceder a páginas web que estén alojadas en otro dominio.
